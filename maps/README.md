# Карты Tarkov

Скачай карты вручную из источников ниже и положи сюда с правильными именами.

## Источники карт

1. **Map Genie** (лучшее качество): https://mapgenie.io/tarkov/maps
2. **Wiki Fandom**: https://escapefromtarkov.fandom.com/wiki/Map_of_Tarkov
3. **tarkov.dev**: https://tarkov.dev/maps

## Имена файлов

- `customs.jpg` — Таможня
- `factory.jpg` — Завод  
- `woods.jpg` — Лес
- `shoreline.jpg` — Береговая линия
- `interchange.jpg` — Развязка
- `reserve.jpg` — Резерв
- `labs.jpg` — Лаборатория
- `lighthouse.jpg` — Маяк
- `streets.jpg` — Улицы Таркова
- `ground-zero.jpg` — Эпицентр
- `sandbox.jpg` — Песочница

## Как скачать

1. Открой Map Genie или Wiki
2. Найди нужную карту
3. ПКМ → "Сохранить изображение как..."
4. Сохрани с правильным именем в эту папку
5. `git add . && git commit -m "Add maps" && git push`
