# Tarkov Bot Images

Публичный репозиторий изображений для Telegram-бота [Tarkov AI Helper](https://t.me/EFTGuidebot).

## Структура

```
tarkov-bot-images/
├── maps/           # Карты локаций с обозначениями
│   ├── customs.jpg
│   ├── factory.jpg
│   ├── woods.jpg
│   ├── shoreline.jpg
│   ├── interchange.jpg
│   ├── reserve.jpg
│   ├── labs.jpg
│   ├── lighthouse.jpg
│   ├── streets.jpg
│   ├── ground-zero.jpg
│   └── sandbox.jpg
├── quests/         # Скриншоты для квестов
│   └── {quest_name}/
│       ├── 1.jpg
│       ├── 2.jpg
│       └── ...
└── README.md
```

## Использование

Все изображения доступны по прямым ссылкам:

```
https://raw.githubusercontent.com/YOUR_USERNAME/tarkov-bot-images/main/maps/customs.jpg
```
